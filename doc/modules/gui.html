Functions

read ()
Read events from the gui.

Returns:

  1. string/number/nil key of element that threw an event
  2. table Values of all elements in the gui, indexed by their index in the widget table
  3. table event table {os.pullEvent()}
new (widgets[, parameters])
Create a new gui object

Parameters:

  • widgets table widget table
  • parameters table (optional)

Tables

gui
Default parameters for gui objects

Fields:

  • disableBuffering Boolean, disable dithering
  • devMode Boolean, devMode enables dragging widgets around and using middle click to gather information about the selected widget.
  • device Not implmented, GUIs can currently only be displayed on the terminal
  • timeout Nil or number, set to enable timeouts
  • autofit Boolean, when enabled the gui will be centered in the middle of the terminal. This expects your gui to be anchored at 1,1.
