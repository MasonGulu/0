Tables

widget
Default widget parameters

Fields:

  • focused bool, is the widget focused?
  • value String, the data contained in the widget, sometimes other types
  • enable_events TOOD, badly implemented at the moment
  • device Device the widget is displayed on, term by default
  • enable bool, render and process events
  • frame bool, draw frame around widget
  • selectable bool, should this object be selectable?
  • theme
theme
Default widget theme, table is contents of widget.theme

Fields:

  • wallLeft char, character used for left side vertical widget walls
  • wallRight char, character used for right side vertical widget walls
  • wallLeftInvert should fg/bg colors be swapped for wallLeft
  • wallRightInvert should fg/bg colors be swapped for wallRight
  • wallLeftFocused char, character used for left side widget walls when focused
  • wallRightFocused char, character used for right side widget walls when focused
  • wallLeftFocusedInvert should fg/bg colors be swapped for wallLeft
  • wallRightFocusedInvert should fg/bg colors be swapped for wallRight
  • frameFG color, text color of frame
  • frameBG color, background color of frame
  • internalFG color, text color of internal widget
  • internalBG color, text color of internal widget
  • internalInvert should fg/bg colors be swapped for internals
