Functions

drawWalls (horizontalChar, verticalChar)
Draw the walls of the box.

Parameters:

  • horizontalChar the character to use for the top and bottom walls, optional defaults to '-'
  • verticalChar the character to use for the left and right walls, optional defaults to '|'

Create a box object.

Parameters:

  • o original object, usually set to nil
  • pos table {x,y}
  • size table {width,height}
  • device the device to display the box on, usually term

Returns:
