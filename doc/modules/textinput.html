Module textinput

A textinput widget.

value is the text input, if in numOnly then the value is converted to a number first. Inherits from the widget object.

Tables

textinput
Defaults for the textinput widget

Fields:

  • type string, used for gui packing/unpacking (must match filename without extension!)
  • default number, default value if numOnly textinput is empty
  • numOnly bool, only accept numbers
  • hasDecimal bool, internal value but if set to true can prevent users from inputing non-integer numbers
  • hideInput bool, whether to hide the input or not
  • enable_events bool, events are enabled by default
